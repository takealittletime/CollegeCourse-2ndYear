• 연결 리스트 연산에 숫자를 정렬하여 삽입하는 애플릿입니다. 삽입하는 숫자에 따라 위치가 결정됩니다.
  • 연결 리스트 연산의 종류를 선택합니다.
  • 삽입할 숫자를 입력난에 입력한 다음, 삽입 버튼을 누르면 삽입 연산이 수행됩니다.
  • 삭제 버튼을 누르면 삭제 연산이 수행됩니다.
  • "수행속도" 버튼을 클릭하여 알고리즘의 수행속도를 선택합니다.(숫자가 높아질수록 속도가 빨라집니다.)
  • 시작 버튼을 누르면 연결 리스트 연산이 시작됩니다.
  • 알고리즘 수행 도중에 수행 속도를 다시 선택할 수 있습니다.
  • 알고리즘을 다시 수행시키고 싶을 때는 정지 버튼을 누르고 다시 시작 버튼을 누르면 됩니다.
