연결된 스택

  • 삽입과 삭제중에서 연산의 종류를 선택합니다.
  • 삽입할 숫자를 입력난에 입력한 다음, 삽입 버튼을 누르면 삽입 연산이 수행됩니다.
  • 삭제 버튼을 누르면 삭제 연산이 수행됩니다.
  • 수행속도를 선택합니다.(숫자가 높아질수록 속도가 빨라집니다.)
  • 다음 버튼을 누르면 버튼을 누를 때마다 단계적으로 실행됩니다.
  • 스택의 크기를 선택합니다. 화면의 크기 제한으로 인하여 16까지로 제한됩니다.
  • 시작 버튼을 누르면 스택 연산이 시작됩니다.
  • 알고리즘 수행 도중에 수행 속도를 다시 선택할 수 있습니다.
  • 알고리즘을 다시 수행시키고 싶을 때는 정지 버튼을 누르고 다시 시작 버튼을 누르면 됩니다.
